# multi-page-application

## 使用方法

1.npm install 或 yarn
2.下载完依赖包之后，你可以在pages文件下创建应用文件夹其目录结构如下
── pages
│   ├── home
│   │   └── index.js \n
|   |   └── app.jsx  \n
|   |   └── index.css
│   ├── login
│   │   └── index.js
        └── app.jsx
        └── index.css

启动服务 npm run dev
打包构建 npm run build
